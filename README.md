# My Blog Project

## プロジェクトの概要
My Blog Project は、Rails をバックエンドに、Vue.js/Vite をフロントエンドに採用したブログ管理システムです。  
ユーザー認証、ブログ記事の作成・編集・削除、記事の検索、カテゴリ・タグによるフィルタリング、ページネーションなどの機能を実装しています。

## プロジェクトのセットアップ手順

### 1. Rails バックエンドのセットアップ
```bash
bundle install
```

### 2.データベースのマイグレーションを実行します：
```bash
rails db:migrate
```


## プロジェクトのセットアップ手順
### 1. 依存パッケージをインストールします：
```bash
npm install
```
# 実装した機能の説明
## ユーザー認証

ログイン、ログアウト機能を実装。ユーザーは認証後に記事の作成や管理が可能です。
## ブログ記事管理

記事の作成、編集、削除機能を実装。
各記事はカテゴリ（Category）とタグ（Tag）と関連付けられており、これらの情報も管理できます。
## 検索機能

バックエンドで実装された search メソッドを利用して、記事タイトルや本文（または抜粋）をキーワードで検索可能です。
フロントエンドでは、検索入力後に Enter キーまたは「Search」ボタンで検索を実行します。
##フィルター機能

カテゴリとタグに基づく記事のフィルタリング機能を実装。
フロントエンド上でドロップダウンメニューから選択し、該当する記事だけを表示します。
## ページネーション

記事一覧はページネーション機能により分割表示され、1ページあたり20件の記事を表示します。
## レスポンシブデザイン

スマートフォンからデスクトップまで、画面サイズに応じたレイアウト調整を行っています。
